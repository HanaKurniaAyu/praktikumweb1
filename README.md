# Apa itu Elemen HTML?
Elemen HTML ditentukan oleh tag awal, beberapa konten, dan tag akhir:
< nama tag > Konten ada di sini... < / nama tag >

Elemen HTML adalah segalanya mulai dari tag awal hingga tag akhir:
​​< p > Paragraf pertama saya. < /p >
